Aplicação de gerenciamento

Aplicação web

Descrição


  • Desenvolvi uma aplicação web dedicada ao gerenciamento de clientes e pedidos. O sistema permite o cadastro eficiente de clientes, bem como o registro de seus pedidos.
  • Pratiquei no projeto conceitos fundamentais da linguagem C# enquanto aprendia a integrá-la com um banco de dados para fornecer funcionalidades como cadastro de clientes e registro de pedidos.
